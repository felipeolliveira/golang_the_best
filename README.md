### 1. Basic Concepts (`_classes/1.basic_concepts`)
Core Go fundamentals with practical examples:
- **Primitives & Types**: Variables, custom types, numeric types, strings (UTF-8/ASCII)
- **Control Flow**: If-else, for loops, switch statements
- **Functions & Data**: Functions, arrays, defer
- **Advanced Basics**: Constants, iota, bitwise operations, imports & packages

### 2. Advanced Concepts (`_classes/2.advanced_concepts`)
Concurrent and parallel programming:
- **Concurrency**: Channels, goroutines, select statements
- **Synchronization**: Mutexes, wait groups, error groups
- **Context**: Context management for cancellation and timeouts

### 3. API Development (`_classes/3.api_rest`)
Building RESTful APIs with Go

### 4. Database Integration (`_classes/4.databases`)
Working with multiple database systems:
- **SQLite**: With CGO and without CGO
- **MySQL** & **PostgreSQL**
- **Tools**: SQLX, SQLC, SQL builders
- **NoSQL**: Redis

- Testing fundamentals and best practices
- Real-world testing examples

### 6. MVC Architecture (`_classes/6.mvc`)
Model-View-Controller pattern implementation
